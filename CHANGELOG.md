# Changelog

## 1.0.2 (2022-02-17)

- Implements `autoCloseTimeout`.
  - You can specify in the Meteor settings, like:
  ```json
  {
    "packages": { 
      "quave:alert-react-tailwind": {
        "autoCloseTimeout": 5000
      }
    }
  }
    ```
